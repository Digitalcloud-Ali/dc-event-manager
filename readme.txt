== Description ==

DC Event Manager is a powerful WordPress plugin that provides a complete solution for managing events on your website. With features like AR integration, carbon footprint tracking, gamification, and Google Maps integration, it offers a unique and engaging experience for both event organizers and attendees.

Key Features:

* Event Creation & Management
* Interactive Event Calendar
* Custom Event Pages
* Shortcodes & Widgets
* Advanced Search & Filtering
* Front-End User Submissions
* Event Notifications & Reminders
* Mobile Responsiveness
* Collaborative Events
* AR Integration for Interactive Experiences
* Event Networking Features
* Carbon Footprint Tracking
* Event Gamification
* Comprehensive Analytics Dashboard
* REST API for External Integrations
* Event Ticketing System
* Recurring Events
* Check-in System
* Google Maps Integration

== Installation ==

1. Upload the `dc-event-manager` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to DC Events > Settings to configure the plugin and enter your Google Maps API key

== Frequently Asked Questions ==

= How do I set up Google Maps integration? =

Go to DC Events > Settings and enter your Google Maps API key in the designated field. Once set, you'll be able to use map features when creating or editing events.

= Can I export event data to other calendar systems? =

Yes, the plugin now supports exporting events to Google Calendar and iCal formats.
